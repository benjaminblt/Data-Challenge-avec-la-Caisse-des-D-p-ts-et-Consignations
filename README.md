# Data-Challenge-avec-la-Caisse-des-D-p-ts-et-Consignations
Projet de data science sur l’évaluation des risques climatiques physiques en France (inondations, sécheresse, feux de forêt, vagues de chaleur). Scoring d’un portefeuille d’actifs via clustering K-means et analyse géospatiale avec QGIS et GeoPandas.
